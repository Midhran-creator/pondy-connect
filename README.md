# Pondy Connect 🏛️
**Developed by: Midhran**

## Overview
Pondy Connect is a transparency dashboard designed for the residents of Puducherry. It addresses the "Dual Power Structure" dilemma by tracking administrative files and public projects across the four enclaves: Puducherry, Karaikal, Mahe, and Yanam.

## Features
- **File Tracker:** Real-time status of major public works.
- **Enclave Localization:** Personalized info for Mahe, Yanam, and Karaikal.
- **Transparency:** Neutral reporting on administrative timelines.

## Tech Stack
- **Language:** Python 
- **Framework:** Streamlit
- **Backend:** Supabase (PostgreSQL)
- pip install streamlit supabase pandas.
